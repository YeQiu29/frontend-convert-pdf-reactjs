## ğŸš€ Instalasi & Menjalankan

Ikuti langkah ini untuk menjalankan proyek di komputer lokal kamu:

### 1. Clone Repository
```bash
git clone [https://github.com/YeQiu29/frontend-convert-pdf-reactjs.git](https://github.com/YeQiu29/frontend-convert-pdf-reactjs.git)
cd frontend-convert-pdf-reactjs
```
### 2. Install Dependencies
Pastikan kamu sudah menginstall Node.js.
```bash
npm install
# atau jika pakai yarn
yarn install
```
### 3. Jalankan Development Server
Pastikan kamu sudah menginstall Node.js.
```bash
npm run dev
```
Aplikasi akan berjalan di http://localhost:5173 (atau port lain yang tersedia).
